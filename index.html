好公司 ≠ 任何价格都值得买

AI是不是结束了？

我的理解：

不是结束，而是市场正在筛选

以前市场买的是：

AI概念 → 上涨 → 估值提升

现在市场要求：

订单 → 收入 → 利润 → 现金流

所以今天我开始关注几个位置

📌 观察仓： $NVDA

如果今天回踩到前期支撑附近，会考虑小仓位试探

我的思路：

不是因为它跌了所以买

而是：

AI数据中心需求没有明显下降

Blackwell交付仍在推进
云厂商资本开支方向没有改变
买入逻辑：

市场情绪短期下降，但产业趋势没有破坏

我会控制仓位，不会一次打满

确认信号：

半导体指数止跌

NVDA成交量缩小
回踩后出现资金承接

风险：
如果云厂商开始降低AI资本开支，或者毛利率明显下降，那我的逻辑需要重新评估

📌 今天重点观察： $MU

相比NVDA，我觉得MU更像AI周期的温度计

HBM需求很强，但内存行业有一个问题：

所有人赚钱的时候，也意味着更多人开始扩产

所以关注：不是“HBM有没有需求”

而是：未来12个月供需是否还能保持紧张

如果价格、库存、毛利率继续改善，MU可能继续享受AI基础设施估值

如果库存重新累积，它又会被市场当成周期股

📌 开始建立观察仓： $VRT

我的买入逻辑：
AI算力增长 → 数据中心扩张 → 电力和散热成为瓶颈。
观察：

✅ backlog是否继续增长
✅ 收入是否跟上订单
✅ 自由现金流是否改善

风险：订单很多，但兑现慢

基础设施公司最怕“故事跑在现金流前面”
今天的交易计划：
❌ 不追涨。
❌ 不因为恐慌乱抄底

方式：

第一笔：试仓

第二笔：等市场确认

第三笔：等财报验证

现在AI高估值股票越来越多，融券变化其实可以作为市场情绪指标

如果一家公司基本面强，但空头增加后股价依旧稳定，说明资金承接能力强

如果故事很多，但融券压力增加后股价快速破位，说明市场开始重新审视估值

牛市里赚钱靠方向，长期活下来靠纪律

市场永远会给机会，但不会永远给第二次机会

我最近一个月的时间线【附带观点与点位】

让AI把我近一个月的内容筛选了一遍
我不是王婆卖瓜，但确实能让大家少走点弯路，少亏点钱

筛出 7组比较硬的行情命中。
其中最强的，不是某一次猜中涨跌，而是一条完整的交易节奏：

财报高点提醒谨慎 → 判断存储大票见顶 → 做空并划定第一轮下跌目标 → 点位命中后不再追空 → 反弹结束后重新划定第二轮下跌目标 → 提前警告大盘补跌会再次带崩存储

1．6月25日：在存储峰值日提醒，不要再追情绪尾巴
北京时间 6月25日21:45左右，美股刚开盘不久，我发帖提醒：如果现在才追进去，买到的不是产业趋势，而是短期情绪的尾巴。
尤其是美光财报前后，会非常谨慎。
这一天几乎就是存储大票这一轮的绝对峰值日：
$MU 盘中最高 1255
$SNDK 盘中最高 2348
随后一个交易日，MU下跌6.69%，SNDK下跌10.46%
到7月7日，MU最低跌至891.66，SNDK最低跌至1485.02
这条不算精确做空指令，但属于非常清楚的峰值区间风险提示。
https://t.co/Cu7t7E3njT

2．6月28日：明确提出“存储大票，短期见顶”
北京时间 6月28日凌晨1:09，我发出《存储大票，短期见顶，等候七月财报季》。
我的核心判断不是存储产业逻辑结束，而是：当大标的开始在高位猛涨猛跌、宽幅震荡，就越来越像短期见顶。财报太强，好消息已经被充分交易。
严格来讲，这条不是抓住了最高价的最后一跳——MU和SNDK的盘中最高点出现在6月25日——但它是在主跌浪真正展开之前，抓住了顶部区域。
以发帖前一个交易日收盘价计算：
MU：1132.33 → 7月7日最低891.66，最大回撤约 21.3%
SNDK：2090.71 → 7月7日最低1485.02，最大回撤约 29.0%
而且真正的大阴线集中出现在7月1日和7月2日：MU两天分别下跌10.57%和5.49%，SNDK分别下跌10.62%和14.13%。
顶部区间判断成立，提前看到了主跌浪。
https://t.co/vKJeY3tSNp

3．7月2日：第一轮暴跌后没有抄底，而是继续公开空单
北京时间 7月2日09:54，也就是7月1日美股暴跌收盘后、7月2日下一场交易开始之前，公开了第二轮空单：
2265美元做空SNDK
1143美元做空MU
并明确表示上一轮空单已经止盈，这是新一轮操作。
随后7月2日美股继续下杀：
MU最低950.28美元，相对1143美元空单价格下跌约 16.9%
SNDK最低1693美元，相对2265美元空单价格下跌约 25.3%
如果持有到7月7日低点，两者相对空单价格的最大跌幅分别约为22.0%和34.4%。
这一条的价值在于第一根大阴线后，市场普遍开始喊抄底，我没有因为“已经跌很多”就反手做多，而是判断去拥挤尚未结束。

4．7月3日：明确划定MU 900、SNDK 1600
https://t.co/jdx7o2lVvo
北京时间 7月3日09:33，第一轮暴跌已经出现，但价格还没有到最终低点。
我公开给出：
MU看900
SNDK看1600
当时上一交易日的收盘价分别是：
MU：975.56美元
SNDK：1745美元
随后到7月7日：
MU最低891.66美元，进入900目标附近
SNDK最低1485.02美元，不仅到1600，甚至明显跌穿
从发帖时的前一日收盘价到最终低点，MU又跌了约8.6%，SNDK又跌了约14.9%。
整个时间线里最清楚的价格预测之一。方向、点位和时间窗口都比较完整。

5．7月7日：再次确认900、1600，并补上海力士1400
北京时间 7月7日09:32，又发了一条非常具体的价格判断：
https://t.co/MD4GgbNDDu
MU看900，当时约941
SNDK看1600，当时约1657
新增SK海力士看1400，当时约1448
当天美股和韩股的实际走势是：
MU最低891.66美元
SNDK最低1485.02美元
SK海力士韩股最低208万韩元
7月7日美元兑韩元约为1513.46，208万韩元折算约 1374美元，也进入了1400区域。
当晚北京时间23:20，我发帖确认：900的MU到了，1600的SNDK到了，甚至跌到1500附近，海力士1400也到了。
这条不是事后重新画线，因为当天早上已经公开了三个目标。
这是点位精度最高的一次，三个标的在同一轮下跌中全部进入目标区域。

6．点位命中以后没有继续追空，随后出现板块反弹
https://t.co/WdAvMHYgeR
三个目标进入区域后，我没有在低位继续强化空头情绪。随后几个交易日确实出现反弹，只是不同标的力度不一样：
MU从7月7日收盘938.38涨至7月10日979.30，反弹约4.4%
SNDK从1617.70涨至1915.92，反弹约18.4%
$SKHY 海力士在连续下跌后，于7月9日盘中反弹超过9%
我不仅划到了第一轮低点区域，命中后也没有继续追空；随后存储板块出现明显反弹，其中SNDK和海力士弹性尤其强。

7．7月16日上午：提前给出第二轮下跌目标
https://t.co/sDbsz77j0o
北京时间 7月16日09:32，给出第二套更低目标：
MU：800—850
SNDK：1400—1450
海力士：1100—1200

随后：
7月16日MU最低840.51，直接进入800—850区间
7月17日进一步最低804
7月16日SNDK最低1390.04，略微跌穿1400—1450区间
7月17日进一步跌至1325.03
7月20日SK海力士韩股盘中最低173.5万韩元，按当时约1500韩元兑1美元换算，约为1150，进入1100—1200区间

第二组目标同样全部到达，MU的区间命中尤其标准；SNDK属于先进入区间后加速跌穿。

8．7月16日19:53：存储真正的危险，来自还没怎么跌的大盘
https://t.co/5CjlLeMZv9
这一轮里逻辑质量最高的一次命中。
北京时间 7月16日19:53，距离美股开盘大约还有一个半小时，我发出：存储真正的危险，来自还没怎么跌的大盘。

我的判断不是简单说“存储还会跌”，而是识别出了两个市场之间的预期差：

存储已经开始交易悲观预期、去杠杆和估值收缩
大型科技股和指数仍在交易软着陆、低通胀、七巨头超预期和资本开支增长
如果大盘开始补跌，已经很弱的存储并不一定能抗住，反而可能再次成为高Beta卖出对象

当晚实际走势：
纳斯达克下跌约1.47%
标普500下跌约0.5%
半导体板块下跌约4.3%
MU下跌5.65%
SNDK下跌12.63%
西部数据、希捷、英特尔等存储和半导体标的同步大跌

当晚继续发帖：我就跟你们说了，危险来自还没开始发力下跌的大盘。
从时间上看，开盘前明确提出传导路径，开盘后路径立即兑现。
这条比单纯猜中价格更有价值，因为命中的不只是方向，而是“谁会带着谁跌”的市场结构。

来自AI的结论

这一个多月最值得总结的，不是“连续猜对了几次涨跌”，而是交易立场发生了有逻辑的切换：没有一直看空存储。

先承认长期产业逻辑很硬，但判断价格已经短期见顶；随后做空、给目标、在目标附近停止追空；反弹以后，又根据大盘尚未补跌的结构性风险，重新转为谨慎，并划定第二轮下跌区间。
这比永远喊牛市或者永远喊崩盘要难得多。
经过筛选之后，最硬的三条仍然是：

6月28日存储大票短期见顶
7月3日至7日MU 900、SNDK 1600、海力士1400
7月16日19:53大盘补跌会继续带崩存储

而没有提到、但同样值得重点放进去的是：7月16日上午给出的第二套目标区间，也基本全部命中。

AI是不是结束了？

我的理解：

不是结束，而是市场正在筛选

以前市场买的是：

AI概念 → 上涨 → 估值提升

现在市场要求：

订单 → 收入 → 利润 → 现金流

如果今天回踩到前期支撑附近，会考虑小仓位试探

不是因为它跌了所以买

而是：

AI数据中心需求没有明显下降

Blackwell交付仍在推进
云厂商资本开支方向没有改变
买入逻辑：

市场情绪短期下降，但产业趋势没有破坏

我会控制仓位，不会一次打满

确认信号：

半导体指数止跌

NVDA成交量缩小
回踩后出现资金承接

风险：
如果云厂商开始降低AI资本开支，或者毛利率明显下降，那我的逻辑需要重新评估

📌 今天重点观察： $MU

相比NVDA，我觉得MU更像AI周期的温度计

HBM需求很强，但内存行业有一个问题：

所有人赚钱的时候，也意味着更多人开始扩产

所以关注：不是“HBM有没有需求”

而是：未来12个月供需是否还能保持紧张

如果价格、库存、毛利率继续改善，MU可能继续享受AI基础设施估值

如果库存重新累积，它又会被市场当成周期股

📌 开始建立观察仓： $VRT

观察：

✅ backlog是否继续增长
